# Insurance Cost Prediction (Machine Learning Project)

This project employs machine learning to predict insurance charges based on factors such as age, BMI, sex, and smoking status.
## Folder Structureinsurance-ml-prediction/
│
├── data
│ └── insurance.csv
│
├── notebook/
│ └── Insurance_Prediction.ipynb
│
└── README.md

## ML Algorithms Used
- Linear Regression
- Lasso Regression
- Polynomial Regression
  

## Best Result
Polynomial regression proved to be the most accurate method.


## How to Use
Run the notebook to execute all data and preprocessing steps.
